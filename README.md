# runpy

 A simple script that executes a python programm everytime it's changed

## Dependencies

python3.x

## Installation

Clone the git repo
```
https://github.com/davidodenwald/runpy.git
```

Make runpy executable
```
cd runpy

chmod +x runpy
```

Move the file so it is in $PATH
```
sudo mv runpy /usr/local/bin/
```

## Usage

```
runpy [FILE]
```

`Ctr-c` to stop runpy
